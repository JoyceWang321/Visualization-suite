# 算法可视化套件 (Algorithm Visualization Suite)

## 一、项目概述

算法可视化套件是一个基于Python开发的桌面应用程序，旨在通过图形化界面直观展示计算机科学中的经典算法。本项目集成了广度优先搜索（BFS）、深度优先搜索（DFS）以及五种排序算法的可视化演示，为算法学习和教学提供了有力的辅助工具。

## 二、功能模块

### 1. BFS算法可视化

• 实现广度优先搜索算法的图形化演示

• 支持交互式图结构生成和节点选择

• 实时显示算法执行过程中的队列状态和访问顺序

• 提供暂停、继续、重置等控制功能

### 2. DFS算法可视化

• 展示深度优先搜索算法的执行流程

• 可视化回溯过程和路径探索

• 支持栈状态监控和路径追踪

• 与BFS算法形成对比分析

### 3. 排序算法可视化

• 集成五种经典排序算法：冒泡排序、选择排序、插入排序、快速排序、归并排序

• 通过柱状图直观展示元素比较和交换过程，采用颜色编码系统区分不同状态

• 支持完整的动画控制：播放/暂停、步进、速度调节、跳转到首尾帧

• 提供算法性能对比分析和实时执行状态显示

• 预计算帧系统：支持前进/后退播放，便于详细观察算法每一步的执行细节

## 三、技术实现

### 1. 开发环境

• 编程语言：Python 3.8+

• 图形界面：Tkinter

### 2. 系统要求

• 操作系统：Windows 10/11, macOS, Linux

## 四、安装与使用

### 1. 环境配置

```bash
# 克隆项目仓库
git clone https://github.com/JoyceWang321/Visualization-suite.git

# 进入项目目录
cd Visualization-suite

# 安装依赖包
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```


### 3. 使用说明

1)启动程序后，在主界面选择要演示的算法模块
   
2)根据需求调整算法参数（节点数量、数据规模等）
   
3)点击相应按钮启动算法可视化
   
4)使用控制面板管理算法执行过程
   
5)观察算法执行结果和分析信息

## 六、算法特性

### 1. 图遍历算法

• BFS算法：时间复杂度O(V+E)，空间复杂度O(V)，适用于最短路径查找

• DFS算法：时间复杂度O(V+E)，空间复杂度O(V)，适用于拓扑排序和连通性检测

### 2. 排序算法

• 冒泡排序：平均时间复杂度O(n²)，稳定排序算法

• 选择排序：平均时间复杂度O(n²)，不稳定排序算法  

• 插入排序：平均时间复杂度O(n²)，适用于小规模数据排序

• 快速排序：平均时间复杂度O(n log n)，分治策略，高效实用

• 归并排序：平均时间复杂度O(n log n)，稳定排序，分治合并策略
